# Employee Schedule Management Application

This project was developed for a Control Structures programming assignment and implements an employee scheduling application in **Python** and **Java**.

The application manages employee schedules for a company that operates seven days a week. It collects employee names and their ranked shift preferences (morning, afternoon, and evening) for each day of the week and generates a final weekly schedule.

## Features

* Collects employee names and daily shift preferences.
* Supports priority ranking of shift preferences.
* Prevents employees from working more than one shift per day.
* Limits employees to a maximum of five working days per week.
* Ensures each shift has at least two employees assigned whenever possible.
* Detects and resolves scheduling conflicts.
* Displays a readable weekly schedule and a summary of days worked.

## Programming Languages

* Python
* Java

## How to Run

### Python

```bash
python employee_schedule.py
```

### Java

```bash
javac EmployeeSchedule.java
java EmployeeSchedule
```

This project demonstrates the use of control structures, including conditionals, loops, branching, input validation, and data structures in two different programming languages.
